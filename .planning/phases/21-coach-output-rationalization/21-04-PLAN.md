# Plan 21-04 - Frontend Types + Rendering Hierarchy + New Components + i18n

## Objective

Update TypeScript types to match the new schema (content_cards, interactive_cards,
transaction_evidence, goal_progress). Implement new rendering components per the
UI-SPEC: ToolUsagePill, TransactionEvidenceTable, GoalProgressBar, ContentCard
(horizontal strip), InteractiveCard (reminder only). Rewrite AssistantBubble rendering
hierarchy to: verdict → message → transaction_evidence → goal_progress → content_cards
→ interactive_cards → details_a2ui (collapsed, STS only). Remove dead components
(LoanCalculatorCard, PartnerOfferCard, GenericActionCard). Add SSE event handling for
tool-usage pills. Add all i18n keys from UI-SPEC copywriting contract.

## must_haves

1. TypeScript types use `content_cards`, `interactive_cards`, `transaction_evidence`, `goal_progress`
2. AssistantBubble renders in correct hierarchy order per UI-SPEC
3. Empty enrichment arrays render no container element (conditional rendering with `&&`)
4. Tool-usage pills appear during streaming as separate messages
5. `new_insight` has no visible rendering (D-05)
6. `details_a2ui` collapsed by default with toggle (D-04)
7. All i18n keys from UI-SPEC added to `it.json` and `en.json`
8. Dead card components removed
9. `pnpm build` clean

    function ContentCardStrip({ cards, locale }: { cards: ContentCard[]; locale: "it" | "en" }) {
      const { t } = useTranslation()
      if (cards.length === 0) return null

      return (
        <div className="mt-3 flex gap-2 overflow-x-auto pb-1 -mx-1 px-1 snap-x snap-mandatory">

          ))}
        </div>
      )

    function ContentCardItem({ card, locale }: { card: ContentCard; locale: "it" | "en" }) {
      const { t } = useTranslation()
      // Icon based on card_type
      const TypeIcon = card.card_type === "video" ? /* Video icon */ :
                       card.card_type === "article" ? /* BookOpen */ :
                       card.card_type === "slide_deck" ? /* Presentation */ :
                       card.card_type === "partner_offer" ? /* Handshake */ :
                       /* Lightbulb for learn */

      const typeLabel = t(`coaching.cardType.${card.card_type}`)
      const isClickable = card.url || card.video_id || card.slide_id

      // For learn cards, show concept + explanation
      if (card.card_type === "learn") {
        return (
          <div className="snap-start shrink-0 w-56 border border-border rounded-lg overflow-hidden bg-card">
            <div className="px-3 py-2 flex items-center gap-1.5 text-xs text-muted-foreground bg-muted/50">
              {/* Lightbulb icon */}
              <span>{typeLabel}</span>
            </div>
            <div className="px-3 py-2">
              <p className="text-sm font-semibold line-clamp-2">{card.concept ?? card.title}</p>
              <p className="text-xs text-muted-foreground line-clamp-2 mt-0.5">{card.plain_explanation ?? card.summary}</p>
            </div>
          </div>
        )
      }

      // For resource-type cards
      return (
        <a
          href={card.url ?? undefined}
          target="_blank"
          rel="noopener noreferrer"
          className="snap-start shrink-0 w-56 border border-border rounded-lg overflow-hidden bg-card hover:border-primary/30 transition-colors"
        >
          <div className="px-3 py-2 flex items-center gap-1.5 text-xs text-muted-foreground bg-muted/50">

          </div>

            <p className="text-sm font-semibold line-clamp-2">{card.title}</p>
            {card.summary && (
              <p className="text-xs text-muted-foreground line-clamp-2 mt-0.5">{card.summary}</p>
            )}
          </div>
        </a>
      )
    }
