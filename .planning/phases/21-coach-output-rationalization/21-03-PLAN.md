# Plan 21-03 - Service Enrichment Pipeline Rewrite + Prompt Updates

Rewrite the coaching service enrichment pipeline to use the new schema field names
(content_cards, interactive_cards), delete `_inject_fallback_cards()` entirely (D-09),
add transaction_evidence extraction (D-07), goal_progress estimation (D-08),
`_validate_a2ui()` quality gate (D-04), apply enrichment caps from Settings (D-10),
and gate enrichments conditionally using tools_called metadata and intent classifier.
Update Jinja2 prompt templates for the new field names and enrichment instructions.

## must_haves

1. `_inject_fallback_cards()` deleted from service.py — no unconditional card injection
2. `_repair_response()` uses `content_cards` and `interactive_cards` field names
3. Caps from Settings applied: content_cards ≤ cap, interactive_cards ≤ cap, evidence rows ≤ cap
4. content_cards empty if `search_content` not in tools_called
5. transaction_evidence populated only if `search_user_transactions` in tools_called
6. goal_progress populated only if conversation is goal-relevant
7. affordability_verdict nullified if intent classifier returns False
8. Jinja2 prompts reference `content_cards`, `interactive_cards`, `transaction_evidence`, `goal_progress`
9. All backend tests pass

    def _gate_enrichments(
        self,
        data: dict,
        tools_called: set[str],
        user_message: str,
        user_id: str | None = None,
    ) -> dict:
        """
        Post-LLM enrichment gating: conditional visibility, caps, quality gates.
        Phase 21 decisions D-03, D-04, D-07, D-08, D-09, D-10.
        """
        from app.coaching.intent import classify_purchase_intent
        from app.core.config import get_settings

        settings = get_settings()

        # D-03: Nullify affordability_verdict on non-purchase intent
        if not classify_purchase_intent(user_message):
            data["affordability_verdict"] = None

        # D-09 + conditional gating: strip content_cards if search_content was not called
        if "search_content" not in tools_called:
            data["content_cards"] = []

        # D-07: Strip transaction_evidence if search_user_transactions was not called
        if "search_user_transactions" not in tools_called:
            data["transaction_evidence"] = None

        # D-08: Strip goal_progress if get_user_preferences was not called
        # (goal progress requires knowing the user's goals)
        if "get_user_preferences" not in tools_called:
            data["goal_progress"] = None

        # D-10: Apply caps
        if len(data.get("content_cards", [])) > settings.coaching_cap_content_cards:
            data["content_cards"] = data["content_cards"][:settings.coaching_cap_content_cards]

        if len(data.get("interactive_cards", [])) > settings.coaching_cap_interactive_cards:
            data["interactive_cards"] = data["interactive_cards"][:settings.coaching_cap_interactive_cards]

        # Cap transaction_evidence rows
        te = data.get("transaction_evidence")
        if te and isinstance(te, dict):
            txns = te.get("transactions", [])
            if len(txns) > settings.coaching_cap_evidence_rows:
                te["transactions"] = txns[:settings.coaching_cap_evidence_rows]

        # D-04: details_a2ui quality gate — strip panels with < 2 data rows
        data = self._validate_a2ui(data)

        # Strip transient metadata before returning
        data.pop("_tools_called", None)

        return data

    def _validate_a2ui(self, data: dict) -> dict:
        """
        Quality gate for details_a2ui: strip if panel has < 2 data rows,
        all-null values, or generic labels.
        Phase 21 decision D-04.
        """
        a2ui = data.get("details_a2ui")
        if not a2ui or not isinstance(a2ui, str):
            return data

        # Parse A2UI JSONL to count data rows
        import json as _json
        lines = [l.strip() for l in a2ui.strip().split("\n") if l.strip()]
        data_rows = 0
        for line in lines:
            try:
                msg = _json.loads(line)
                # dataModelUpdate messages contain actual data rows
                if msg.get("type") == "dataModelUpdate":
                    updates = msg.get("updates", [])
                    data_rows += len(updates)
            except _json.JSONDecodeError:
                continue

        # Strip if fewer than 2 data rows
        if data_rows < 2:
            data["details_a2ui"] = None
            logger.debug("details_a2ui stripped: only %d data rows", data_rows)

        return data
    ```

    ```
    - **transaction_evidence**: Mini-table of user transactions that support your advice. Populate ONLY when you cite specific transactions from `search_user_transactions` results. Copy the date, description, and amount from the tool results verbatim. Maximum 5 rows. Null if no transactions cited.
    ```

    ```
    - **goal_progress**: Progress toward a user goal. Populate ONLY when the conversation relates to one of the user's stated goals (from `get_user_preferences`). Example: goal_name="Fondo di emergenza", estimated_pct=40, subtitle="Stimato dal tuo margine mensile attuale". Null if no goal is relevant.
