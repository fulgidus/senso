```jinja2
### A2UI Component Reference

The `details_a2ui` field must be a string containing newline-separated JSON protocol messages.
Always emit three message types in order: surfaceUpdate → dataModelUpdate → beginRendering.

**Available components:**
- `text`: {"type": "text", "id": "...", "value": "..."}  — plain text block
- `card`: {"type": "card", "id": "...", "title": "...", "children": [...]}  — titled content card
- `textField`: {"type": "textField", "id": "...", "label": "...", "value": "..."}  — labelled data field (for exact numbers)
- `timeline`: {"type": "timeline", "id": "...", "items": [{"label": "...", "description": "..."}]}  — ordered steps
- `button`: {"type": "button", "id": "...", "label": "...", "action": "..."}  — action button (renders, not yet wired)

**Protocol structure:**
```json
{"type": "surfaceUpdate", "surface": {"type": "card", "id": "root", "title": "TITLE", "children": [COMPONENTS]}}
{"type": "dataModelUpdate", "data": {}}
{"type": "beginRendering"}
```

**Example — expense breakdown:**
```json
{"type": "surfaceUpdate", "surface": {"type": "card", "id": "root", "title": "Analisi spesa", "children": [{"type": "textField", "id": "income", "label": "Reddito mensile netto", "value": "€ 1.850"}, {"type": "textField", "id": "expenses", "label": "Spese ricorrenti", "value": "€ 1.230"}, {"type": "textField", "id": "margin", "label": "Margine disponibile", "value": "€ 620"}]}}
{"type": "dataModelUpdate", "data": {}}
{"type": "beginRendering"}
```

Emit `null` for `details_a2ui` when the answer is simple and no structured breakdown adds value.
```
