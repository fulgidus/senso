<task id="20-06-02" title="Italy rules content coverage verification">
  <objective>
    Assert that `italy_rules.json` has sufficient coverage for the top 10 financial
    questions likely asked by Italian young adults.
  </objective>
  <read_first>
    - api/app/content/italy_rules.json
    - api/app/content/search.py — search_italy_rules()
  </read_first>
  <action>
    ```python
    @pytest.mark.parametrize("query,expected_topic", [
        ("IRPEF aliquote", "irpef"),
        ("bonus cultura 18app", "bonus"),
        ("INPS contributi", "inps"),
        ("730 dichiarazione", "730"),
        ("TFR liquidazione", "tfr"),
        ("ISEE calcolo", "isee"),
        ("partita iva forfettario", "partita-iva"),
        ("affitto cedolare secca", "affitti"),
        ("fondo pensione", "pensione"),
        ("bonus psicologo", "bonus"),
    ])
    def test_italy_rules_coverage(query, expected_topic):
        results = search_italy_rules(query, locale="it")
        assert len(results) >= 1
        topics = [r["id"] for r in results]
        assert any(expected_topic in t for t in topics)
    ```
  </action>
  <acceptance_criteria>
    - All 10 parametrized queries return ≥1 result
    - `pytest test_italy_rules.py` — 0 failures
  </acceptance_criteria>
