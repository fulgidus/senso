# Plan 39-01: personal_finance Chest Schemas

## Goal

Write the flat JSON Schema files for all 7 personal_finance Chests. These schemas drive `DomainModelFactory` table generation (Phase 34). Schemas must be flat (no nested objects, no arrays) — each field maps to a typed column.

## must_haves

- All 7 schema files exist: user_profile, transactions, coaching_insights, timeline_events, content, regional_knowledge, user_prefs
- Every schema is flat (zero `"type": "object"` or `"type": "array"` properties)
- `user_prefs` schema has: `first_name`, `last_name`, `voice_gender`, `default_persona_id`, `voice_auto_listen` — all previously on platform `users` table
- `transactions` schema has: `amount`, `date`, `merchant`, `category`, `description`, `source_document_type`, `currency`, `is_income`
- `user_profile` schema has: `income_total`, `monthly_margin`, `employment_status`, `main_currency`
- `content` schema has: `title`, `summary`, `type`, `slug`, `body`, `topics_csv` (topics as CSV string — flat alternative to array)
- `manifest.yaml` updated: all 7 chest `schema_ref` fields point to these files
- `DomainModelFactory.build_tables(manifest)` succeeds without `SchemaFlattenError`

## Verification

```bash
# All schemas valid and flat
docker compose run --rm api python -c "
import json
from pathlib import Path

for schema_file in Path('domains/personal-finance/chests').glob('*.schema.json'):
    schema = json.loads(schema_file.read_text())
    for prop_name, prop in schema.get('properties', {}).items():
        assert prop.get('type') not in ('object', 'array'), \
            f'{schema_file.name}: property {prop_name!r} is type {prop[\"type\"]!r} — must be flat'
    print(f'✓ {schema_file.name}: {len(schema.get(\"properties\", {}))} flat fields')
"

# DomainModelFactory builds all 7 tables
docker compose run --rm api python -c "
from pathlib import Path
from sqlalchemy import MetaData
from app.domain.loader import load_manifest
from app.data.factory import DomainModelFactory
m = load_manifest(Path('domains/personal-finance/manifest.yaml'))
tables = DomainModelFactory(MetaData()).build_tables(m, Path('domains/personal-finance'), lambda: 'k')
expected = {'user_profile','transactions','coaching_insights','timeline_events','content','regional_knowledge','user_prefs'}
assert set(tables.keys()) == expected, f'Missing: {expected - set(tables.keys())}'
print(f'✓ All 7 domain tables built correctly')
```
