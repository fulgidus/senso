      ```python
      def _record_trace(self, db, upload_id, step_order, step_name, *, status="success",
                        input_summary=None, output_summary=None, raw_input=None, raw_output=None,
                        duration_ms=None):
          """Write a single trace row. Never raises - failures are logged and swallowed."""
          import time
          from app.db.models import IngestionTrace
          try:
              trace = IngestionTrace(
                  upload_id=upload_id,
                  step_name=step_name,
                  step_order=step_order,
                  input_summary=input_summary,
                  output_summary=output_summary,
                  raw_input=(str(raw_input)[:8000] if raw_input is not None else None),
                  raw_output=(str(raw_output)[:8000] if raw_output is not None else None),
                  duration_ms=duration_ms,
                  status=status,
              )
              db.add(trace)
              db.commit()
          except Exception as exc:
              import logging
              logging.getLogger(__name__).warning("_record_trace failed: %s", exc)
      ```
    - In `run_extraction_background()` (the background task wrapper), wrap the call to the pipeline with 5 trace points:
      1. step_order=1, step_name="start", status="success", input_summary=f"file: {upload_id}"
      2. step_order=2, step_name="ocr_extraction", capturing file_bytes length in input_summary, result summary in output_summary
      3. step_order=3, step_name="module_match", input_summary=extracted doc type, output_summary=module_name
      4. step_order=4, step_name="llm_call", input_summary=prompt snippet (first 500 chars), output_summary=raw_output snippet
      5. step_order=5, step_name="persistence", output_summary="saved to DB"
    - Use try/except around each trace so failures never break ingestion
    - Capture start = time.time() before each step, compute duration_ms = int((time.time() - start) * 1000)
    - Each _record_trace uses a FRESH db session (or the passed-in db) - use self.db directly; do NOT hold a session open across multiple separate commits

    IMPORTANT: The instrumentation wraps existing calls without changing their behavior. If any step doesn't have a clean hook point, add the trace just before and just after calling the step. Use the existing db session from IngestionService (self.db).

      @router.get("/ingestion/uploads/{upload_id}/trace", response_model=list[IngestionTraceDTO])
      def get_upload_trace(
          upload_id: str,
          _: UserDTO = Depends(require_admin),
          db: Session = Depends(get_db),
      ):
          from app.db.models import IngestionTrace, Upload
          # Verify upload exists (no ownership check - admin sees all)
          upload = db.query(Upload).filter(Upload.id == upload_id).first()
          if not upload:
              raise HTTPException(status_code=404, detail="Upload not found")
          traces = (
              db.query(IngestionTrace)
              .filter(IngestionTrace.upload_id == upload_id)
              .order_by(IngestionTrace.step_order)
              .all()
          )
          return [IngestionTraceDTO.model_validate(t) for t in traces]
