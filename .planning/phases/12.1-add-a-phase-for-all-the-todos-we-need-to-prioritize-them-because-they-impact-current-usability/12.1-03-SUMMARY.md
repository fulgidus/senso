## Decisions Made

- **Hook separation principle**: `useAuth.ts` `signOut` must not render UI. The `window.confirm` was incorrectly placed in a hook. Moved confirmation to the call-site components (SettingsScreen). This is the correct architectural separation.
- **FilesTab uses id-based state**: `deleteTargetId: string | null` (not boolean) to track which specific file is being deleted, since the delete action needs the file ID.
- **Radix umbrella package**: `radix-ui` (umbrella v1.4.3) already in package.json includes `@radix-ui/react-dialog`. No additional install needed; dialog.tsx imports via `"radix-ui"`.
