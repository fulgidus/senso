Fix five admin content table UX issues (sort affordance, icon headers, pagination, locale grouping, missing translations) plus two profile usability fixes: localized category names in the uncategorized picker (#9) and save confirmation / reset controls in SettingsScreen (#8).

Output: Paginated admin table with proper sort UX, localized category picker, and improved settings save UX.
