## Current Test

Hotfixes applied after first prod UAT run (2026-04-02). Three failures found and fixed:

1. **PWA topbar still shows** - fixed: added `id`, `scope`, `display_override` to `manifest.webmanifest`. Requires app re-install on device.
2. **Pull-to-refresh broken** - fixed: removed pull-to-refresh from ChatScreen entirely; added top-30%-viewport guard in `usePullToRefresh`; removed `touchAction: "pan-x"` overrides from ChatScreen and ProfileScreen.
3. **TTS fallback amber badge not showing on Firefox/Librewolf** - fixed: `useTTS` now exposes `hasFallenBack` (persistent, pre-set on browsers without SpeechRecognition). `VoicePlayButton` uses `hasFallenBack` for the badge instead of the transient `usingFallback`.
