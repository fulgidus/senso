<objective>
Four UI polish fixes: convert the spending breakdown bar chart to a pie chart (#12), make all data tables responsive on mobile with card layouts (#16), add a visible TTS fallback indicator when ElevenLabs is unavailable (#22), and fix coaching card dark mode rendering (#23).

Purpose: The bar chart works but a pie chart is better for showing spending distribution at a glance — it matches the "at-a-glance" financial profile intent. Tables on mobile currently require horizontal scroll, which is hard to use. ElevenLabs TTS relies on an API key that may not be configured in all environments; users need clear feedback that voice is running in degraded mode. LearnCardStub uses hardcoded blue colors that are unreadable in dark mode.

Note on #22 (TTS): The root cause is `ELEVENLABS_API_KEY` being empty in `api/.env`. This plan adds a visible fallback indicator in the UI — a small amber badge on the voice button when `usingFallback=true`. The API key must be configured separately by the developer. This plan does NOT modify the backend.

Output: Pie chart in ProfileScreen, responsive card tables in all admin/ingestion pages, TTS fallback badge, dark-mode coaching cards.
</objective>

**Fix #22 — TTS fallback indicator in ChatScreen.tsx:**

Find the `TTSButton` component (around line 570-598). Currently `usingFallback` is only shown in the `title` attribute. Add a visible amber dot/badge when fallback is active:

```tsx
function TTSButton({ text, locale, ttsConfig }: { ... }) {
  const { t } = useTranslation()
  const { canPlay, isPlaying, isGenerating, usingFallback, play, stop } = useTTS(ttsConfig)
  const busy = isGenerating || isPlaying

  if (!canPlay) return null

  return (
    <div className="relative inline-flex items-center">
      <Button
        type="button"
        variant="ghost"
        size="icon"
        className="h-6 w-6 text-muted-foreground hover:text-foreground"
        onClick={() => (busy ? stop() : void play(text, locale))}
        disabled={isGenerating && !isPlaying}
        aria-label={isGenerating ? t("coaching.ttsGenerating") : isPlaying ? t("coaching.ttsPlaying") : t("coaching.ttsPlay")}
        title={
          usingFallback ? t("coaching.ttsFallbackHint") :
          isGenerating ? t("coaching.ttsGeneratingShort") :
          isPlaying ? t("coaching.ttsPlayingShort") :
          t("coaching.ttsPlayShort")
        }
      >
        {isGenerating ? (
          <Loader2 className="h-3 w-3 animate-spin" />
        ) : isPlaying ? (
          <Square className="h-3 w-3" />
        ) : (
          <Volume2 className="h-3 w-3" />
        )}
      </Button>
      {usingFallback && (
        <span
          className="absolute -top-1 -right-1 h-2 w-2 rounded-full bg-amber-400"
          title={t("coaching.ttsFallbackHint")}
          aria-hidden="true"
        />
      )}
    </div>
  )
}
```
