  "backendUrl": "http://localhost:8000",
  "frontendUrl": "http://localhost:3000"

Required endpoint contracts to define in this plan:
```text
POST /auth/signup    body { email, password } -> 201 { user, accessToken, refreshToken, expiresIn }
POST /auth/login     body { email, password } -> 200 { user, accessToken, refreshToken, expiresIn }
GET  /auth/google/start -> 200 { authUrl } or 503 { fallback: "email_password" }
GET  /auth/google/callback?code=...&state=... -> token payload or fallback error
POST /auth/refresh   body { refreshToken } -> 200 { accessToken, refreshToken, expiresIn }
GET  /auth/me        header Authorization: Bearer <accessToken> -> 200 { user }
POST /auth/logout    body { refreshToken } -> 204
```
</interfaces>
