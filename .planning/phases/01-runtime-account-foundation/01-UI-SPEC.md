## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage                     |
| ----- | ----- | ------------------------- |
| xs    | 4px   | Icon gaps, inline padding |
| sm    | 8px   | Compact element spacing   |
| md    | 16px  | Default element spacing   |
| lg    | 24px  | Section padding           |
| xl    | 32px  | Layout gaps               |
| 2xl   | 48px  | Major section breaks      |
| 3xl   | 64px  | Page-level spacing        |

Exceptions: interactive tap/click targets must be at least **44px** in both dimensions.

## Color

### Dark Theme

| Role            | Value   | Usage                                                                          |
| --------------- | ------- | ------------------------------------------------------------------------------ |
| Dominant (60%)  | #1B262C | App background and primary surfaces                                            |
| Secondary (30%) | #0F4C75 | Cards, auth panels, nav regions                                                |
| Accent (10%)    | #3282B8 | Primary CTA, focus rings, active auth mode indicator, inline status highlights |
| Destructive     | #BBE1FA | Destructive actions only                                                       |

### Light Theme

| Role            | Value   | Usage                                                                          |
| --------------- | ------- | ------------------------------------------------------------------------------ |
| Dominant (60%)  | #F9F7F7 | App background and primary surfaces                                            |
| Secondary (30%) | #DBE2EF | Cards, auth panels, nav regions                                                |
| Accent (10%)    | #3F72AF | Primary CTA, focus rings, active auth mode indicator, inline status highlights |
| Destructive     | #112D4E | Destructive actions only                                                       |

Accent reserved for: primary CTAs, keyboard focus outlines, active auth-tab/chip state, and high-priority inline notices. Do not use accent for all interactive elements.

## Copywriting Contract

Tone: **direct, supportive**.

| Element                  | Copy                                                                                      |
| ------------------------ | ----------------------------------------------------------------------------------------- |
| Primary CTA              | Create account                                                                            |
| Empty state heading      | You’re not signed in yet                                                                  |
| Empty state body         | Create an account or continue with Google to access your financial coaching workspace.    |
| Error state              | We couldn’t complete sign-in right now. Try again, or use email and password to continue. |
| Destructive confirmation | Sign out: Are you sure you want to sign out on this device?                               |
